# ZW-finShop. Заявка на кредит Онлайн  
## Проектная работа

### I.  Модуль "Банк-клиент"

1. Клиент вводит ПД: 
    * Имя 
    * Паспорт
    * Доход
2. Клиент вводит запрашиваемую сумму и отправляет заявку
3. Заявка создается в статусе "Ожидает решения"
4. Через некоторое время статус заявки изменяется (автоматически или по запросу) в зависимости от принятого Банком решения 

### II. Модуль "Кредитный менеджер"

1. Менеджер видит Журнал Заявок в виде таблицы
2. Менеджер кликает ссылку в строке новой заявки (в статусе "Ожидает решения") и открывает карточку "Заявка ID"
3. Менеджер запрашивает необходимые данные о Клиенте, оценивает параметры заявки и заполняет доп. информацию (одобренная сумма)
4. Менеджер переводит заявку в статус "Одобрено" или "Отказано"

### III. Модуль "Процессинговый центр"

1. БД Postgres
2. Шина Kafka (для интеграции с Кредитным Менеджером)
3. Веб Интерфейс GRPC (для интеграции с Банк-Клиентом)
