### I.  Модуль "Банк-клиент"

1. Клиент вводит ПД:
    * Имя
    * Паспорт
    * Доход
2. Клиент вводит запрашиваемую сумму и отправляет заявку
3. Заявка создается в статусе "Ожидает решения"
4. Через некоторое время статус заявки изменяется (автоматически или по запросу) в зависимости от принятого Банком решения
